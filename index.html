Política de Privacidade

Última atualização: 21 de junho de 2025

1. Coleta de dados

Não coletamos dados pessoais sensíveis. As únicas informações que tratamos são:

  • Dados de uso anônimos: métricas de desempenho e telas acessadas, coletados para diagnóstico e melhoria do aplicativo.
  • Dados de rede: parâmetros de requisição para buscar pontuações e estatísticas públicas sobre partidas.

2. Uso de dados

Os dados coletados são utilizados exclusivamente para:

  • Exibir placares ao vivo e estatísticas detalhadas.
  • Enviar notificações de início de partidas e resultados.
  • Otimizar a experiência do usuário a partir de análises agregadas.

3. Compartilhamento de dados

Não compartilhamos nenhum dado com terceiros. Todas as informações de uso são mantidas de forma agregada e anônima.

4. Armazenamento e segurança

Utilizamos conexões seguras (HTTPS) para todas as requisições. Não armazenamos informações pessoais no dispositivo do usuário.

5. Cookies e tecnologias similares

O aplicativo não utiliza cookies, mas pode usar armazenamento local do dispositivo para cache de dados e configurações.

6. Suporte e contato

Se tiver dúvidas ou desejar mais informações, entre em contato:

  • E-mail: suporte@arena360x.com.br
  • Website: arena360x.github.io

Para detalhes sobre termos de uso, consulte nossos Termos de Serviço.
